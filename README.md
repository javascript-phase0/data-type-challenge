# data-type-challenge

Contoh 1:
let kata = 'Stefani'

maka output adalah 'username Stefani'

Contoh 2:
let kata = 25

maka output adalah 'age 25'

Contoh 3:
let kata = 0

maka output adalah 'Invalid data'

Contoh 4:
let kata = '0'

maka output adalah 'username 0'

Contoh 5:
let kata = false

maka output adalah 'cannot proceed without agreement'
